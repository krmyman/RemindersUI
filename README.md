# 待辦事項 App

這是一個依照 `需求.md`、`UI.md`、`流程.md`、`元件.md` 與 `實作計畫.md` 落成的 React + Vite 前端專案。

## 功能

- 三個頁面：`全部`、`進行中`、`已完成`
- 新增待辦事項
- 刪除待辦事項
- 標記待辦事項完成
- 預設深色模式
- 桌機與手機皆可操作

## 啟動方式

先確認本機已安裝 Node.js 18 以上版本，再執行：

```bash
npm install
npm run dev
```

## 目前範圍

此版本只包含前端介面與本地狀態，不包含：

- 登入
- 後端 API
- 雲端同步
- 搜尋
- 排序
- 標籤
- 提醒
